Intro#

Como seres vivos para obtener información, hacemos uso de datos y los alojamos en nuestra mente, es decir organizamos los datos en nuestra base de datos (la mente) y los moldeamos internamente, es decir, damos forma al interior o dicho de otra forma nos in-formamos para aprender y obtener conocimiento. A la hora de clasificar la información hacemos lo mismo ya sea en un entorno físico con un folio o uno digital con dispositivo electrónico. En un entorno físico para clasificar la información escribimos en cuadernos, cartapacios, pizarras; mientras en un entorno digital necesitamos herramientas que se adapten.

Para organizar digitalmente la información que abunda y a veces excede es fundamental seguir un buen método de categorización o workflow y contar las herramientas necesarias para facilitar el proceso. Así lograremos un sistema ordenado más fácil para consultar y estudiar, incluso por personas que conozcan del tema.

El método de categorización es un asunto personal sin embargo el software empleado es universal para todos. A este tipo de software enfocado a la organización de archivos se les denomina gestores o managers (en inglés). Según el uso hay varios tipos, en este artículo se citarán los más relevantes.

Gestor de backups#

Los gestores de copias de seguridad son esenciales para mantener sincronizados los archivos entre dispositivos y fuentes de almacenamiento. Par así evitar la pérdida de archivos y mantener los dispositivos actualizados al día.

  • FreeFileSync. Uno de los mejores software de comparación y sincronización de carpetas (multiplataforma, open-source y gratuito), sirve para crear y gestionar copias de seguridad de los archivos.
  • Duplicati. Un cliente de copia de seguridad que almacena de forma segura copias de seguridad cifradas, incrementales y comprimidas en el almacenamiento local, servicios de almacenamiento en la nube y servidores de archivos remotos.
  • RClone. Como rsync pero en la nube, es un programa de línea de comandos para sincronizar archivos y directorios desde y hacia diferentes proveedores de almacenamiento en la nube.
  • Syncthing. App abierta, fiable y descentralizada. Los datos se almacenan de forma privada en un el lugar de almacenamiento seleccionable.
  • Time Machine. Una conocida app de Mac que permite hacer copias de seguridad y retroceder en el tiempo a ese estado de la máquina.

También se pueden sincronizar archivos mediante comandos en la terminal:

  • robocopy. Es un comando de Windows que permite hacer copias de seguridad entre dos rutas de directorios.
  • rsync. Una utilidad de código abierto que proporciona una rápida transferencia incremental de archivos, es completamente gratis para descargar.
  • cp. Comando de Linux para copiar archivos y directorios. ´cp [OPTION] filename destination Directory´. [Option]=-f,-h,-i,-n,-p,-r, –backup.

Gestor de archivos#

Los gestores de archivos son los software más antiguos y los más conocidos para organizar todo tipo de archivos, suelen ser visores generales y editores de metadatos de archivos. No suelen ser compatible con los archivos ya que precisan de reproductores, editores, etc. Suelen ser en un entorno gráfico o en su forma más sencilla listados de archivos en la terminal de comandos.

Con entorno gráfico#

Para Android (móvil)#

  • Cx Explorer, Oppo File Manager, Solid Explorer, FX File Explorer, X-plore File Manager.

Para Linux#

  • 4 pane. 4Pane es un gestor de archivos multipanel y con listas detalladas para Linux.
  • Caja - Escritorio Mate
  • Gestor de archivos DDE
  • DMenu
  • Delfín - KDE. El gestor de archivos más completo del escritorio KDE Plasma.
  • Lfm - Katxi. Gestor de archivos de Lfm.
  • Files - GNOME. Gestor de archivos de GNOME.
  • Nemo - Cinnamon. Gestor de archivos de Cinnamon.
  • Krusader. Gestor de archivos de doble panel para su escritorio.
  • PCManFM. Un gestor de archivos extremadamente rápido y ligero que cuenta con una navegación por pestañas y una interfaz fácil de usar.
  • Polo file manager. Polo es un moderno y ligero gestor de archivos para Linux con soporte para múltiples paneles y pestañas; soporte para archivos, y mucho más.
  • Qtfm.eu. Gestor de archivos básico para linux.
  • XFCE Thunar. Thunar es un moderno gestor de archivos para el entorno de escritorio Xfce.

Para Mac#

  • Finder el gestor por defecto de la manzana en el PC.
  • Commander One](https://mac.eltima.com/file-manager.html). Gestor de archivos de doble panel gratuito para Mac
  • ForLift 3. El más avanzado gestor de archivos de doble panel y cliente de transferencia de archivos para macOS.
  • HoudahSpot. HoudahSpot es una herramienta esencial para los profesionales y usuarios avanzados que trabajan con miles de archivos. HoudahSpot recoge las agujas del proverbial pajar.
  • Buscador de rutas.
  • Transmit. Una aplicación de transferencia de archivos para macOS, sube, descarga y gestiona archivos en toneladas de servidores con una interfaz de usuario fácil, familiar y potente.

Para Windows#

  • Navegador (Chromium, Firefox) con la ruta del directorio: file:///C:/
  • Explorador de archivos de Windows. El explorador de archivos de Windows es una de las aplicaciones más viejas y conocidas de Microsoft para gestionar archivos en local. Poco ha cambiado desde sus inicios y a pesar de ello sigue siendo muy útil, lo que indica el buen diseño desde su inicio. Permite ver y organizar todo tipo de archivos con múltiples vistas, ordenar por filtros y ver las carpetas con estructura de árbol con las rutas de directorios. La desventaja es que no hay etiquetado ni mejoras en el tiempo.
  • Files Windows. Gestor de archivos parecido al de Windows pero con mejor interfaz gráfica, open-source y etiquetado por colores.
  • XYplorer. Gestor de archivos con múltiples opciones y formatos de vista para Windows.
  • TagSpaces. Gestor de archivos con etiquetado propio (incluye geoetiquetas), mediante dos métodos: cambiando el nombre o añadiendo un json en un base de datos propia.

Multiplataforma#

  • Navegador web: Chromium (Google Chrome, Brave, MS Edge, Opera, Vivaldi), Firefox (Librewolf). leer y abrir archivos con la ruta en el navegador: file:///C:/ .
  • TagSpaces. Gestor de archivos con etiquetado propio (incluye geotags), por dos métodos: cambiando el nombre o añadiendo un json en una base de datos propia.

Con comandos desde la terminal#

Para Linux#

  • Cfiles. Un gestor de archivos ncurses escrito en C con combinaciones de teclas similares a las de vim.
  • Lf. Gestor de archivos de terminal escrito en Go con una fuerte inspiración en el gestor de archivos Ranger.
  • Midnight Commander. GNU Midnight Commander es un gestor de archivos ortodoxo multiplataforma gratuito.
  • nnn. El gestor de archivos de terminal poco ortodoxo con muchas características.
  • ranger. ranger es un gestor de archivos de consola con enlaces de teclas VI.
  • Vfim.info. Vifm es un gestor de archivos con interfaz curses, que proporciona un entorno similar a Vim para la gestión de objetos dentro de los sistemas de archivos, ampliado con algunas ideas útiles de mutt.
  • WCM Commander. Gestor de archivos de código abierto gestor de archivos ortodoxo.

Para Windows#

  • Windows terminal with commands (cd, dir, attrib, rename, tree).

Gestor de librerías#

Los gestores de librerías tiene un uso general para categorizar y visualizar casi todo tipo de formatos, son los más amplios en cuanto a visualización propia.

  • Adobe Bridge. Adobe Bridge es un potente gestor de activos creativos que permite previsualizar, organizar, editar y publicar múltiples activos creativos de forma rápida y sencilla. Edite los metadatos. Añada palabras clave, etiquetas y clasificaciones a los activos. Organice los activos mediante colecciones y encuentre activos mediante potentes filtros y funciones de búsqueda avanzada de metadatos. Óptimo para la suite de Adobe, aunque se precisa de cuenta y es de pago mensual.
  • Eagle.cool. El mejor gestor de archivos enfocado para diseño de imágenes pero con un uso ampliable a todo tipo de archivos. La interfaz de usuario es muy elegante y bien estructurada, en lo visual hay dos paneles laterales con información y en el medio el listado de archivos.
    • En el panel izquierdo está la parte estructural, con tres bloques:
      1. Etiquetado general con 7 secciones (“Todo, Sin categorías, Sin etiquetado, Reciente, Aleatorio, Gestor de etiquetas, Papelera”.
      2. Carpetas inteligentes y acceso directo de carpetas.
      3. Listado de carpetas en árbol.
      4. Cifrado de carpetas con contraseña.
    • En el panel derecho se encuentra los metadatos asociados a cada archivo (Miniatura, color similar, nombre del archivo, etiquetado, notas, url, carpetas, propiedades (puntuación, dimensiones, duración, tamaño, tipo, fecha de creación y fecha de modificación)
    • En la parte central se muestra el listado de archivos que permite encontrar archivos por filtros, múltiples opciones para ordenar y botón para realizar acciones para un conjunto de archivos de etiquetado. Los archivos se pueden ordenar según filtros personalizables y cuenta con múltiples tipos de visión de archivos en pantalla.
    • La parte negativa es que es de pago y solo sirve para dos dispositivos. Y su librería de almacenamiento solo es visible con la info asociada por Eagle.

Plataforma online#

  • Gestores DAM: Digitile.io, Damion, Canto, Infinity, Tagbox.io, Taglery.
  • Gestores de la nube: Dropbox, Google Drive, MEGA, Nexcloud, pCloud.com, Sync.com
  • Gestor de archivos en plataformas de chat: WhatsApp, Telegram, Discord.
  • Gestores de contenido online: metabuscadores (Google, DuckDckGo, Bing), plataformas de audio (Spotify), imagen (Pixabay), texto (Openlibrary), video (YouTube).
  • Gestor de proyectos: Notion.so

Gestores por formato#

Gestor de audio (reproductor de música)#

Todos los gestores de audio mencionados a continuación son gratuitos.

  • MusicBee. El mejor reproductor de música de Windows, muy personalizable, con edición de metadatos e información adicional de canciones.
  • MusicBrainz Picard. App de auto-etiquetado con los metadatos de canciones de música (artistas, álbumes, compositores, etc).
  • Foobar2000. Un viejo conocido reproductor de música que permite organizar los archivos de audio en Windows.
  • AIMP. Gestor de audio con una interfaz renovada y ligera.
  • Audacious. Gestor de audio ligero, multiplataforma y de código abierto.
  • Strawberry music player. Es un reproductor de música ligero, multiplataforma y de código abierto.

Gestor de imágenes (galería)#

Los gestores de imágenes suelen ser galerías de imágenes con opciones adicionales.

  • ACDSEE Photo Studio Home. Gestor de Activos Digitales (DAM) de pago (90 $ anual). Incluye:
    • Modo “Multimedia”, que ahorra tiempo y permite interactuar directamente con la base de datos de fotos de ACDSee.
    • Modo “Personas”, un motor de IA que coloca sin problemas un nombre a una cara. Una perspectiva vibrante y colorida le espera con las mejoras de Home 2022 en las Curvas de Tono y la compatibilidad con ISOs más altos.
  • Alussion. Herramienta construida para artistas, cuyo objetivo es ayudarte a organizar tu biblioteca visual. Un único lugar que contiene toda tu colección de referencias, inspiración y cualquier otro tipo de imágenes.
  • Digikam. App para gestionar fotografías, cuenta con reconocimiento de caras y etiquetado de imágenes.
  • KPhotoAlbum. Gestor de imágenes de KDE con sistema de etiquetado.
  • LibrePhotos. Servicio de gestión fotográfica de código abierto auto-alojado, orientado para instalación en Docker.
  • Photoprism. Similar a LibrePhotos usa una IA para navegar, organizar y compartir tu colección de fotos. Cuenta con sistema de etiquetado y puede ser usada en local, servidor privado o en la nube. Orientado para instalación en Docker.
  • Photoview. Galería de fotos para servidores personales.
  • Xnview MP. Gestor de imágenes con galería integrada y edición de fotos.

Gestor de texto#

Los gestores de texto sirven para navegar entre los archivos asociados a directorios y editar su contenido. Se suelen usar para crear código y editar documentación.

Editores de texto o de desarrollo integrado (IDE)#

Permiten ver directorios enteros con todos sus archivos y editar el código en texto plano. Además cuentan con un compilador y más características.

  • Visual Studio Code. Visual Studio Code es el editor para desarrolladores más conocido, es gratuito pero su código es privado perteneciente a Microsoft. Tiene multitud de plugins y funcionalidades.
  • VSCodium. Un Visual Studio pero de código abierto y sin envío de datos en segundo plano.
  • Notepad++. Editor gratuito famoso, liviano y simple.
  • Sublime Text. Editor de texto de pago pero que puede ser usado gratis de por vida.
  • Netbeans. IDE famoso de Apache orientado a java, php, html.
  • Emacs. Editor de texto multiplataforma.
  • NeoVim. Editor del famoso Vim pero con entorno renovado.
  • Atom.io. Editor orientado para GitHub con Git.
  • KDevelop. Editor de la comunidad KDE.
  • Brackets. Editor para desarrollo web y previsualización de markdown.
  • Eclipse.org. Editor orientado para C++.
  • IntelliJ IDEA. Editor de pago de IntelliJ, con una interfaz lograda.

Editores markdown (gestor de notas)#

Los editores de formato markdown (md) son muy útiles para gestionar la información y extraer conocimiento. Además su formato de texto plano lo hace muy versátil para cualquier aplicación y dispositivo.

  • Obsidian.d. App para gestión de notas en formato markdown, cuenta con un panel lateral en vista de árbol y múltiples opciones de búsqueda y etiquetado. También permite reproducir multitud de archivos desde audio (mp3, wav), imágenes (jpg, png, bmp, tiff), videos (mp4, mkv, mov) y documentos (pdf). Tiene un visor gráfico y multitud de plugins variados, con una estética simple y elegante personalizable hasta el último detalle.
  • Logseq. App open source gratuita para manejo de archivos markdown (md), similar a Obsidian pero las notas se almacenan en un archivo común.
  • Joplin. Al igual que las anteriores Joplin sirve para gestionar archivos markdown, tiene una buena interfaz y botones para editar en formato markdown. A diferencia de los anteriores no permite visualizar el contenido como grafo ni añadir archivos distintos de markdown.

Gestor de vídeos#

Hay pocas apps que se dediquen exclusivamente a gestión de videos, ya que suele ir incluido en gestores de archivos, galería o gestores de bibliotecas. Y las pocas que hay están enfocadas hacia la pornografía aunque este uso es simplemente opcional.

  • Porn Vault. Es un gestor de imágenes y videos multiplataforma para enfocado pornografía, pero sirve para cualquier tipo de videos. Permite etiquetar, añadir información adicional como la persona que aparece, el estudio, puntuación.

  • Stash. Califica tus escenas e imágenes, y etiquétalas con intérpretes, etiquetas, películas y estudios. Ordena tu contenido con una variedad de opciones de filtrado y clasificación. Stash también permite obtener metadatos a partir de los nombres de los archivos y cuenta con un pequeño editor de video incorporado. El software es gratuito, de código abierto y privativo, el programa corre en una red local privada visible a través del navegador y permite cifrar con contraseña.

  • Vide Hub App. Es un gestor de videos gratuito con una interfaz que muestra varias miniaturas y referencias de los videos agregados.

  • En móvil (Android o iOS) los gestores por formato son la galería (imágenes y vídeos) y los reproductores de música que permiten clasificar el audio, algunos gestores de notas como Obsidian son también compatibles.

    • Galería: Gallery Yerapps, Simple Gallery, Oppo Gallery, Samsung Gallery, Xiami Gallery.

El gestor de archivos “ideal”#

La app perfecta para gestión de archivos debería contar con las siguientes características:

  1. Licencia: código abierto (para ser transparente frente al uso de datos).
  2. Coste: gratuito (con donativo) o pago único de por vida (precio máximo 80 €).
  3. Plataformas: múltiples (Linux, Mac, Windows, iOS, Android).
  4. Privacidad: app local privada (sin envío de datos a terceros).
  5. Interfaz: simple con variedad de temas (claro y oscuro).
  6. Sistema de clasificación: estructura jerárquica con carpetas y ruta de directorios.
  7. Clasificación general:
    1. Almacenamiento que conserve la estructura de directorios.
    2. Etiquetado de archivos, puntuación, adición de metadatos.
    3. Filtrado de archivos por tipo o metadatos.
    4. Guardado de páginas webs (con opción de lectura offline).
    5. Muestra de archivos con distintas vistas (detalles, iconos, lista, miniatura).
    6. Ordenar por metadatos de archivos (nombre, fecha de creación, etiqueta…).
    7. Panel de búsqueda avanzado.
    8. Panel de ruta del directorio actual.
    9. Visores gráficos: estructura de árbol y visor de grafos.
  8. Formatos compatibles.
    1. Formatos reproducibles:
      • Audio (aac, aiff, dsd, flac, mp3, wav, wma).
      • Documentos (csv, docx, epub, html, md, pdf, txt, xml).
      • Imagen (bmp, gif, jpg, png, psd, raw, svg, tiff, web).
      • Video (avi, divx, flv, mp4, mov, mkv, webm).
    2. Formatos editables (opcional):
      • Audio, edición de velocidad, recorte y tono.
      • Documentos, edición en texto plano.
      • Imagen edición de contraste, brillo, saturación, sombras y color.
      • Videos edición de valores de audio + imagen.
  • El gestor “ideal” sería una mezcla de:
    • Eagle.cool por el mostrado de archivos, interfaz y etiquetado en grupos.
    • Explorador de Windows o Files por su filtrado y opciones organización.
    • Obsdian.md por su visor de grafos, interfaz personalizable y plugins compatibles.
    • TagSpaces por su geoetiquetado, permitiendo conservar la estructura de carpetas.

Mi preferido hasta la fecha es Eagle.cool pero el problema es que no permite abrir todos los archivos internamente (aunque para eso están los editores) y su almacenamiento es en una librería propia del programa con una carpeta por archivo con (nombre.xyz + nombre_minuatura.xyz + nombre.json) lo que hace que la transferencia de archivos sea mucho más lenta por la cantidad de carpetas creadas. Otro punto en contra es que solo permite dos dispositivos con registro en Windows o Mac. Otro gestor muy útil es Files la versión moderna y open-source del conocido explorador de Windows, cuenta con etiquetado de color, multiventana y variedad de temas. Lo malo es que solo es de Windows, la visión de carpetas es jerárquica y el etiquetado es básico sin agrupación ni posibilidad de exportar.

Como dato para añadir metadatos a los archivos se precisa de una base de datos externa, ya que normalmente los archivos según el formato vienen con un límite de metadatos propio. Esto hace que se tengan que añadir archivos con información extra (info-extra.json) y que la estructura de carpetas cambie. Una forma para conservar la estructura del directorio es añadir los metadatos en el nombre del archivo pero este es un método limitado.

Conclusión#

El software utilizado para gestionar archivos es una buena herramienta para clasificar la información y facilitar el trabajo de recolección y aprendizaje. Pero a fin de cuentas lo que importa es el objetivo que quieras conseguir con esos datos, el método lo puedes ir variando hasta dar con el que más te guste o con el que te resulte más cómodo.
