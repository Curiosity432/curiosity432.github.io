Las distribuciones de Linux y los entornos de escritorio más importantes

Introducción#

Para empezar en Linux lo primero que tienes que hacer antes de instalar el sistema operativo es decidir qué distribución de Linux (distro) quieres usar y con qué entorno de escritorio, estos son dos conceptos que si quieres sumergirte en Linux tienes que entender. Desde la creación del Kernel de Linux se necesitaron más componentes de software para instalar el sistema operativo como un agujero, para facilitar el proceso de instalación del kernel nacieron las distribuciones de Linux, piezas de software que vienen con un gestor de paquetes permitiendo instalar aplicaciones fácilmente sin hacer la instalación manualmente compilando desde el código fuente cada app.

En este artículo, vamos a hablar de estos dos conceptos: las distribuciones Linux y los entornos de escritorio Linux. La principal diferencia es que la distro es el sistema operativo con software para funcionar correctamente y el entorno de escritorio es el software para mejorar la experiencia del usuario con GUI^[GUI: Graphical User Interface].

Distribuciones Linux#

Las distribuciones de Linux son básicamente la combinación de:

  • Núcleo Linux. El núcleo del sistema operativo (SO).
  • Herramientas y bibliotecas GNU, por lo que a veces se llama GNU/Linux.
  • Gestor de paquetes, herramientas de software que se encargan de instalar, actualizar y eliminar aplicaciones en el ordenador. Los gestores de paquetes instalan paquetes de software desde repositorios en línea, para su uso es necesario escribir los comandos específicos en el terminal, por ejemplo, “pacman -Syu” para actualizar los paquetes en Arch Linux.
  • Entornos de escritorio. La mayoría de las distribuciones de Linux incluyen un entorno de escritorio para facilitar el uso del sistema pero esto no es imprescindible por ser una distribución. Se pueden instalar varios DE (entornos de escritorio) en la misma distribución.

La principal diferencia entre distribuciones es el gestor de paquetes utilizado, cada distro tiene su propio gestor de paquetes y están enfocadas a un tipo de uso: para ordenadores personales, para servidores, para smartphones, para dispositivos electrónicos (dispositivos embebidos, smartwatches, routers…).

Cuando aparece una nueva distro basada en una original con código bifurcado, se conoce como distro derivada y si la misma distro tiene diferentes entornos de escritorio para instalar, se conoce como sabores de la distribución.

Hay cientos de distribuciones de Linux pero la mayoría son distros derivadas basadas en:

  • Arch Linux => ArchCraft, ArcoLinux, Black Arch, EndeavourOS, Garuda Linux, Manjaro, XeroLinux

  • Basadas en Debian => Ubuntu (ElementaryOS, Linux Mint, PopOS, Lubuntu, Xubuntu, Zorin), Kali Linux, MX-Linux, Parrot, PureOS, Tails, Raspberry OS, SparkyLinux

  • RHEL (Red Hat Entriprise Linux) => Fedora, CentOS, AlmaLinux, RockyLinux, ClearOS

  • Gentoo => Sabayon Linux, Calculate Linux, Redcore Linux, Gentoo Studio

  • Slackware => KateOS, SlixOS

  • Otras distros: Void Linux

Cuando una distro Linux utiliza diferentes isos para instalarse con diferentes entornos de escritorio, estas distros se llaman sabores de la distro X. Por ejemplo: Ubuntu por defecto instala el entorno de escritorio GNOME, Kubuntu viene con KDE (K Desktop Environment), lo mismo con Fedora con GNOME y Fedora Spins (con KDE, XFCE, LXQT…), y muchas otras distribuciones.

Entornos de escritorio#

  • Los entornos de escritorio (ED) son los programas que se ejecutan sobre el sistema operativo de un ordenador y que comparten una interfaz gráfica de usuario común, el propósito es ayudar al usuario a trabajar en el ordenador fácilmente. Sin embargo, la interfaz de línea de comandos tradicional tiene más características.

Aquí hay una lista de ellos, KDE y GNOME son los escritorios más famosos.

  • DE: KDE, GNOME, XFCE, Budgie, Deepin DE, Mate desktop, LXDE, Lumina

  • El entorno de escritorio incluye:

    • Administración del sistema / Sistema de inicio (systemd)
    • Sistema de ventanas (Xorg (X11), Wayland)
    • Gestor de ventanas
      • Gestor de ventanas flotantes (budgie, cinnamon, cutefish, cwm, deepin, dusk, gnome, icewm, jwm, KDE plasma, leftwm, lxde, lxqt, openbox, ukui, xfce4)
      • Gestor de ventanas en mosaico (awesome wm, bspwm, dwm, Herbstlufwm, i3 wm, qtile, spectrwm, wmderland, xmonad)
    • Gestor de inicio de sesión (ssdm, lightdm, gdm, lxdm, ly)
    • Menú (dmenu, rofi)
    • Tema global (cursor, barra de tareas, tema, fondo de pantalla)
    • Lanzador de aplicaciones (rofi)
    • Aplicaciones GUI (navegador, compresor, gestor de archivos, visor de imágenes, reproductor de audio, reproductor de vídeo, visor de pdf, editor de texto, editor de oficina, grabadora)

Una cosa a mencionar es que los entornos de escritorio no son necesarios para trabajar en el ordenador, sin embargo la distro de Linux es obligatoria. Para sustituir el DE puedes crear el tuyo propio instalando el software que quieras, ejemplo de sistema de flujo de trabajo: Awesome WM + dmenu + rofi.
