Enlaces#

Mostrar enlaces internos#

  • Añadir imágenes desde una carpeta dentro del directorio “static” creado por HUGO (img, jpg, gif)
![Test](/<route-path>#center)
{{ <figure src="/img/test/Test.jpg#center" height="100" width="200"> }} <!--sin espacios-->

  • Añadir vídeos desde una carpeta dentro del directorio “estático” creado por HUGO (mp4, webp, ogg)
{\{< rawhtml >}}
<video width=100% controls autoplay>
    <source src="/videos/test/sample-video.mp4" type="video/mp4">
    Su navegador no soporta la etiqueta de vídeo.    
</video>
{\{< /rawhtml >}}

Mostrar enlaces externos#

  • Iframe se suele usar para mostrar contenido de la web, pero no funciona con HUGO <iframe src="https://www.youtube.com/embed/vQYiGRConFU"></iframe>

{/{< youtube vQYiGRConFU >}/}

  • Shrotcodes para añadir cosas (paréntesis juntos):
{ {< video "http://clappr.io/highline.mp4" "my-5" >}}
