Intro#

To begin in Linux the fisrst thing you have to do before installing the operating system is to decide which Linux distribution (distro) do you want to use and with which desktop environment, this are two concept that if you want to dive into Linux you need to understand. Since the creation of the Linux Kernel more software components were needed to install the operating systema as a hole, to facilitate the installation process of the kernel were borned the Linux distributions, pieces of software that comes with a package manager permitting to install applications easily without doing the installation manually compiling from source every app.

In this article, we are going to talk about these two concepts: the Linux distributions and Linux desktop environments. The main difference is that the distro is the operating system with software to work properly and the desktop environment is the software to improve the experience of the user with GUI^[GUI: Graphical User Interface].

Linux distros#

The Linux distributions are basically the combination of:

  • Linux Kernel. The core of the operating system (OS).
  • GNU tools and libraries, for that reason sometimes is called GNU/Linux.
  • Package manager, software tools that deal with installing, upgrading and removing apps on the computer. The package managers install software packages from online repositories, for use it you need to type the specific commands on the terminal, e.g. “pacman -Syu” to update the packages on Arch Linux.
  • Desktop environments. The majority of Linux distributions include a desktop environment to facilitate the use of the system but this is not imprescindible for being a distribution. You can install multiple DE (desktop environments) in the same distribution.

The main difference between distributions are the package manager used, each distro has their own package manager and are focused on a type of use: for personal computers, for servers, for smartphones, for electronic devices (embedded devices, smartwatches, routers…).

When a new distro appears based on an original one with forked code, it is known as a derived distroand if the same distro has different desktop environments to install, it is known as flavors of the distribution.

There are hundreds of Linux distributions but most of them are derived distros based on:

  • Arch Linux => ArchCraft, ArcoLinux, Black Arch, EndeavourOS, Garuda Linux, Manjaro, XeroLinux

  • Debian based => Ubuntu (ElementaryOS, Linux Mint, PopOS, Lubuntu, Xubuntu, Zorin), Kali Linux, MX-Linux, Parrot, PureOS, Tails, Raspberry OS, SparkyLinux

  • RHEL (Red Hat Entriprise Linux) => Fedora, CentOS, AlmaLinux, RockyLinux, ClearOS

  • Gentoo => Sabayon Linux, Calculate Linux, Redcore Linux, Gentoo Studio

  • Slackware => KateOS, SlixOS

  • Other distros: Void Linux

When a Linux distro uses differents isos to install with differents desktop environments, this distros are called flavours of X distro. For example: Ubuntu by default install GNOME desktop enviroment, Kubuntu comes with KDE (K Desktop Environment), same with Fedora with GNOME and Fedora Spins (with KDE, XFCE, LXQT…), and many others distributions.

Desktop environments#

  • The desktop environment (DE) are the programs running on top of a computer operating system that share a common GUI, the purpose is to help the user to work on the computer easily. However the traditional command-line interface has more features.

Here is a list of them, KDE and GNOME are the desktop more famouse.

  • DE: KDE, GNOME, XFCE, Budgie, Deepin DE, Mate desktop, LXDE, Lumina

  • The desktop environment includes:

    • System administration / Init system (systemd)
    • Window system (Xorg (X11), Wayland)
    • Window manager
      • Floating window manager (budgie, cinnamon, cutefish, cwm, deepin, dusk, gnome, icewm, jwm, KDE plasma, leftwm, lxde, lxqt, openbox, ukui, xfce4)
      • Tiling window manager (awesome wm, bspwm, dwm, Herbstlufwm, i3 wm, qtile, spectrwm, wmderland, xmonad)
    • Login manager (ssdm, lightdm, gdm, lxdm, ly)
    • Menu (dmenu, rofi)
    • Global theme (cursor, taskbar, theme, wallpaper)
    • App launcher (rofi)
    • GUI Apps (browser, compressor, file manager, image viewer, audio player, video player, pdf viewer, text editor, office editor, recorder)

One thing to mention is that desktop environments are not necessary to work on the computer however the Linux distro is obligatory. To replace DE you can create your own installing the software you want, example of workflow system: Awesome WM + dmenu + rofi.
