  • Block code:

`Inline code` with one back quote (`) at the beggining and at the end.

Three tildes (~~~) or back quotes (```) at the beggining and at the end of the block.
~~~
Block code                   [x] **This should be a bold text**
~~~

Links#

Show internal links#

  • Add images from a folder inside “static/” directory created by HUGO (img, jpg, gif)
![Test](/<route-path>#center)
{{ <figure src="/img/test/Test.jpg#center" height="100" width="200"> }} <!--without spaces-->

  • Add videos from a folder inside “static” directory created by HUGO (mp4, webp, ogg)
{\{< rawhtml >}}
<video width=100% controls autoplay>
    <source src="/videos/test/sample-video.mp4" type="video/mp4">
    Your browser does not support the video tag.  
</video>
{\{< /rawhtml >}}

Show external links#

  • Iframe to show content on the web but doesn’t work with HUGO <iframe src="https://www.youtube.com/embed/vQYiGRConFU"></iframe>

{/{< youtube vQYiGRConFU >}/}

  • Shrotcodes to add things (brackets together):
{ {< video "http://clappr.io/highline.mp4" "my-5" >}}
